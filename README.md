# Vibe Coding

一个基于 Python `tkinter` 的俄罗斯方块小游戏。

## 项目简介

本项目实现了一个可直接运行的经典俄罗斯方块，包含：

- 方向键移动与旋转
- `↓` 加速下落
- 空格硬降
- `P` 暂停 / 继续
- `R` 结束后重新开始
- 计分、等级、下一块预览

## 运行方式

确保你的电脑已安装 Python 3，然后在项目根目录执行：

```bash
python AI/tetris.py
```

## 操作说明

- `←` / `→`：左右移动
- `↑`：旋转
- `↓`：软降
- `空格`：硬降
- `P`：暂停 / 继续
- `R`：游戏结束后重新开始

## 项目结构

- `AI/tetris.py`：游戏主程序
- `AI/requirements.txt`：依赖说明（本项目无第三方依赖）

## 说明

本项目使用标准库 `tkinter`，无需额外安装第三方包。
